Sequence Analysis Toolkit

The Sequence Analysis Toolkit is a tool designed for biochemists to make sequence analysis quick and easy

Current state

The Sequence Analysis Toolkit is currently under active development. The front end is mostly complete and the flask back end can be extended quickly to add new functions. The backend currently supports Blast protein searches only.

Intentions

The Sequence Analysis Toolkit will in future be available on the web and as a standalone desktop app with access to a python shell allowing both the speed and ease of the visual interface and the power of python. Cloud storage for sequences and workflows will allow work to be synced between the desktop app and the web app allowing sequence analysis to be performed on any computer with a browser and internet connection and continued on a machine with python.
